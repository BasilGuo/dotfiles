> This repo is for dotfiles.

# dotfiles

## For Bash
> DIR: ~/, running `. ~/.vimrc` after modification.
- `.bashrc`
- `.colors.sh`
- `.git_prompt.sh`

## For Git
> DIR: ~/
- `.gitalias`
- `.gitconfig

## For Vim
> DIR: ~/
- `.vimrc`
- `.gvimrc`

## For Tmux
> DIR: ~/.config/tmux/
- `.tmux.conf`

## For XTerm
> DIR: ~/, running `xrdb -merge ~/.Xresources` after modification.
- `.Xresources`
