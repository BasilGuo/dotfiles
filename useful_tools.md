# Useful Tools

## command line

### tldr
too long; don't read.
It is a substitute of `man`.

```shell
$ sudo apt install -y tldr

$ tldr <command>
```
You could also use the `npm` to install this.

### tmux
terminal multiplexer.

```shell
$ sudo apt install -y tmux

$ tmux list-session
$ tmux ls
$ tmux new-session -s
$ tmux attach
$ tmux attach -t
```
